Stripe Setup Guide

1Create Stripe Account

Sign up for a Stripe account at stripe.com

2Get API Keys

Navigate to Dashboard → Developers → API keys. Copy your Publishable key and Secret key.

3Configure Webhook

Create webhook endpoint in Stripe Dashboard → Developers → Webhooks

Webhook URL: https://yourdomain.com/api/stripe-webhook
Events: checkout.session.completed, payment_intent.payment_failed

4Environment Variables

STRIPE_PUBLISHABLE_KEY=pk_live_your_key_here
STRIPE_SECRET_KEY=sk_live_your_key_here
STRIPE_WEBHOOK_SECRET=whsec_your_webhook_secret

Supabase Setup Guide

1Create Project

Sign up at supabase.com and create a new project

2Get Credentials

From project settings, copy: Project URL, API Key (anon/public), Service Role Key

3Create Tables

CREATE TABLE donations (
id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
amount DECIMAL(10,2) NOT NULL,
donor_email TEXT,
status TEXT DEFAULT 'pending'
);

4Environment Variables

SUPABASE_URL=https://your-project.supabase.co
SUPABASE_ANON_KEY=your_anon_key_here

OpenAI Setup Guide

1Get API Key

Sign up at platform.openai.com and create a new secret key

2Set Usage Limits

Set monthly usage limits to control costs. Start with $10-20/month for testing.

3Environment Variables

OPENAI_API_KEY=sk-your_api_key_here

Email Marketing Setup

1Choose Provider

Select: Mailchimp, ConvertKit, SendGrid, or Mailgun

2Get API Credentials

Generate API key from your provider's dashboard and verify sending domain

3Configure Environment

SENDGRID_API_KEY=SG.your_api_key_here
SENDGRID_FROM_EMAIL=noreply@yourdomain.com
